# Сайт компании ДезДело

Готовый адаптивный многостраничный сайт на HTML + CSS + JS.

## Страницы:
- index.html — главная (+ счётчики чисел)
- prices.html — цены
- reviews.html — отзывы
- faq.html — FAQ (с аккордеоном)
- contacts.html — контакты
- services/ — услуги (5 страниц)

## Дополнительно:
- style.css — полный стиль со всеми анимациями
- faq.js, counters.js, countUp.min.js — JS-скрипты
- sitemap.xml, robots.txt — для SEO

Инструкция: распакуйте и залейте содержимое на хостинг.
